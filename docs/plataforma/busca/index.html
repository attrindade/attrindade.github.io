    Funcionalidade de Busca

    Introdução

    A funcionalidade de busca da plataforma GovernaClima permite ao usuário pesquisar por diversas entidades e informações contidas no banco de dados.

    Para a pesquisa de dados governamentais em nível federal, é possível pesquisar por:

    • Organizações
    • Iniciativas
    • Atos Normativos
    • Consultas e Audiências Públicas

    Para as pesquisas de dados governamentais em nível estadual, é possível pesquisar por:

    • Atos Normativos

    Para a pesquisa de dados da sociedade civil em nível nacional, possível pesquisar por:

    • Organizações
    • Iniciativas

    Busca por pessoas

    A busca por pessoas será implementada nas próximas atualizações.

    Abaixo estão descritos os detalhes de cada funcionalidade e filtro disponíveis para facilitar a pesquisa.

    Tela de Busca

    Barra de Pesquisa

    A barra de busca é o principal ponto de interação do usuário com o sistema de pesquisa. Nela, o usuário pode selecionar uma categoria principal e aplicar filtros opcionais para refinar os resultados.

    Passos para realizar uma busca:

    1. Selecione a Categoria: O usuário deve escolher a categoria de pesquisa entre "Organização", "Iniciativas" ou "Atos Normativos".

    2. Filtros Opcionais: Além da categoria principal, o usuário pode aplicar filtros opcionais, como:

      • Filtrar por tipo: Permite selecionar o tipo específico de entidade dentro da categoria escolhida.
      • Filtrar por variável: Habilita filtros adicionais com base nas variáveis específicas de cada categoria.
      • Filtrar por data: Define um intervalo de datas para refinar os resultados.

    3. Busca por Termos: O usuário pode inserir termos de busca no campo de texto, para realizar uma busca por nome dentro dos registros selecionados.

    4. Botão "Buscar": Após selecionar os filtros e/ou inserir os termos de busca, o usuário deve clicar no botão "Buscar" para exibir os resultados.

    Categorias e Filtros

    Organizações

    • Tipo de organização: Permite escolher entre os tipos disponíveis para filtrar os resultados, como Tipo de organização.

    • Variáveis Disponíveis:

      • Org. coordenadora ou envolvida: Filtra organizações que atuam como coordenadoras ou envolvidas.
      • É instrumento institucional?: Filtra se a organização é um instrumento institucional.
      • Em andamento?: Permite filtrar organizações que ainda estão em andamento.

    • Data de Início e Fim: O usuário pode definir um intervalo de datas de início e fim das organizações.

    Iniciativas

    • Tipo de iniciativa: Permite filtrar os resultados com base no tipo de iniciativa.

    • Variáveis Disponíveis:

      • Organização coordenadora: Filtra iniciativas com base na organização coordenadora.
      • Em andamento?: Filtra se a iniciativa está em andamento ou foi concluída.

    • Data de Início e Fim: Permite filtrar iniciativas com base em um intervalo de datas específicas.

    Atos Normativos

    • Tipo de ato normativo: Permite selecionar o tipo de ato normativo que deseja buscar.

    • Variáveis Disponíveis:

      • Status: Filtra atos normativos com base no status atual.
      • Esfera: Permite filtrar pela esfera do ato.
      • Poder: Filtra com base no poder que criou ou emitiu o ato normativo.

    • Data de Criação: O usuário pode definir um intervalo de datas de criação dos atos normativos.

    Consultas e audiências públicas serão documentadas em breve

    Resultados da Pesquisa

    Visualização dos Resultados

    Após realizar a busca, os resultados serão exibidos em formato de tabela, contendo as seguintes colunas:

    • Nome: Nome da entidade correspondente ao filtro.
    • Outras Informações: Dependendo da categoria, pode exibir número, tipo, status, entre outros.
    • Ações: Um link "Acessar" para detalhes específicos da entidade.

    Caso não haja resultados para a pesquisa realizada, a mensagem "Nenhum resultado encontrado" será exibida.

    Exemplos de uso

    Exemplo de uso

    Exemplo: Busca de Organizações por Nome

    1. Categoria: Organizações
    2. Filtro aplicado: Tipo de Organização → "Organizações Públicas"
    3. Filtro variável: Em andamento → "Sim"
    4. Busca por termo: Inserir "Ambiental"

    Neste exemplo, a busca retornará todas as organizações públicas em andamento cujo nome contenha o termo "Ambiental".
